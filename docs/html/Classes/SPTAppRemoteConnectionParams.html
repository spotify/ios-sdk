Overview

The SPTAppRemoteconnectionParams represents connection parameters required in order to initiate a connection with the Spotify app.

  clientIdentifier

A unique identifier for the client.

@property (nonatomic, copy, readonly) NSString *clientIdentifier

Discussion

The client key is used for authentication, and must be obtained from the Spotify developer portal.
